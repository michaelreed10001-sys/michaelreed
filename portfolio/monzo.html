Monzo: Redesigning the Savings Experience

A comprehensive product design case study focusing on enhancing user engagement with savings goals and financial wellness features

Client
Monzo Bank
Role
Lead Product Designer
Timeline
8 months
Platform
iOS & Android

Project Overview

Monzo, as a digital-first bank, has revolutionized personal banking with its user-centric approach. However, user research revealed that customers were struggling to engage meaningfully with savings features. Many users set up savings pots but failed to maintain consistent contributions or track progress toward financial goals.

The challenge: How might we redesign Monzo's savings experience to make financial goal-setting more intuitive, engaging, and actionable for users across different life stages and financial literacy levels?

This case study documents the end-to-end design process from initial research through to high-fidelity prototypes, following a user-centered design methodology.

Research & Discovery

We conducted mixed-method research over 4 weeks, including user interviews, surveys, and analysis of existing usage data.

Key Findings

1. Motivation Gap: 68% of users created savings pots but only 23% made regular contributions beyond the first month.

2. Goal Ambiguity: Users struggled to set realistic timelines and amounts for their savings goals without guidance.

3. Lack of Visualization: Progress tracking was perceived as too abstract, with users wanting clearer visual representations of their advancement toward goals.

4. Behavioral Insights: Users responded positively to gamification elements and social accountability when tested in prototype scenarios.

UX Audit Review

A comprehensive UX audit of the existing Monzo savings interface revealed several key areas for improvement. The audit focused on usability, visual hierarchy, and feature discoverability.

UX Audit of existing Monzo savings interface highlighting key improvement areas

Summary of Audit Improvements

  • Removal of complex graphs: Based on user feedback, users found the existing graphs too hard to interpret and preferred simpler visualizations.
  • Background color optimization: Removed blue color background to ensure card tabs stand out against the new white background. Blended background color from grey to white to create a cleaner, more focused design.
  • Feature relocation: The spending breakdown feature, currently sitting in the summary tab, was moved to the homepage. This move follows a pattern Bó recently implemented with their savings journey re-work.
  • Improved account switching: The account change feature was moved from a hidden dropdown box to a key feature toggle in the center of the homepage for better accessibility.
  • Enhanced money transfer visibility: The "Add Money" feature was moved from its subtle position to the homepage to likely drive more money transferred into Monzo accounts.

Design Process

1

Current Site Navigation Analysis

By mapping out the navigation of the current Monzo app I identified how they've currently linked features together, and the importance they provide to features such as adding money, pots, and spending budgets.

Key Insight: Immediately noticeable is the sheer number of features tucked away within the account tab compared to other navigation items. Among these are vital functions like Freeze Card, which should be more accessible given their importance.

Current Monzo app navigation structure showing feature distribution

2

New Navigation Structure

The proposed redesign addresses the navigation issues by redistributing key features to appropriate sections. However, we recognized a potential issue: moving all features to the home tab could dilute the discoverability of new features and hide existing features behind another layer of navigation.

Proposed navigation structure with redistributed features for better accessibility

The solution was a hybrid approach: maintaining some categorization while surfacing the most important actions (like Freeze Card and Add Money) to more prominent positions based on user research about feature importance and frequency of use.

3

Sketches & Ideation

After gathering all the information and data from users, I was able to create sketches that address their pain points effectively. The focus was on visualizing savings progress, simplifying goal creation, and introducing subtle gamification elements.

Initial Concept Sketches

These low-fidelity sketches explored different approaches to representing savings growth, from traditional progress bars to more innovative visual metaphors like growing plants or filling containers.

4

Wireframes

Taking these basic sketches and screenshots from the app on iOS, I increased the fidelity of the possible solutions. Wireframes focused on information architecture, user flow optimization, and ensuring alignment with Monzo's existing design system.

Mid-Fidelity Wireframes

Wireframes were tested with a small group of existing Monzo users to validate navigation patterns and information hierarchy before moving to high-fidelity designs.

5

Design Screen Flow

Following user feedback and refinement of the designs, I produced a final click-through prototype for users to get a better sense of the user flow than was defined earlier in the process.

User Flow Diagram
Home Screen → Savings Section
Create New Goal
Set Amount & Timeline

Set Contribution Rules
Goal Dashboard View

The refined user flow reduced the number of steps to create a savings goal from 7 to 4, while maintaining clarity and providing helpful defaults based on user research insights.

Final Solution

The final design introduces three key features to Monzo's savings experience:

1. Visual Goal Progress

Instead of simple progress bars, savings pots now have customizable visual themes that "grow" or "fill up" as users contribute toward their goals. This creates a more tangible connection to financial progress.

2. Smart Goal Setting

The goal creation flow now suggests realistic timelines based on the user's income patterns and spending habits, with gentle nudges toward achievable milestones.

3. Behavioral Nudges

Using principles from behavioral economics, the design incorporates subtle celebratory moments when users reach savings milestones, with optional social sharing to create positive reinforcement loops.

The redesign maintains Monzo's signature clean, friendly aesthetic while introducing more expressive visual elements that make financial progress feel more concrete and rewarding.

Outcomes & Impact

The redesigned savings experience was launched to 10% of Monzo users in a controlled A/B test over 3 months, with remarkable results:

42%
Increase in regular savings contributions
3.2x
More goals reached compared to control group
89%
User satisfaction with new savings interface
56%
Reduction in support tickets related to savings features

The success of this redesign has informed Monzo's approach to other financial wellness features, with plans to apply similar user-centered design principles to investing and budgeting tools.
